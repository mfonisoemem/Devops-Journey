# Devops-Journey
 🎯 Goal
To develop a solid understanding of the DevOps ecosystem, with a strong emphasis on **cost awareness** both for cloud 
and on-premise deployments to make smarter, data-driven business decisions.
The ultimate aim is to **bridge the gap between technical execution and strategic project management**, ensuring that technology decisions align with business goals, timelines, and budgets.
 
## **Mission**
My mission over the next 6 months is to gain a deep, hands-on understanding of DevOps and AWS — not just in theory, but through consistent practice, real-world projects, and cloud deployments.
Although I have prior exposure to these areas, this time I’m diving in with full commitment to:
- Build real systems from scratch
- Understand the DevOps lifecycle end-to-end
- Confidently manage software projects with firsthand technical context
- Bridge the gap between Project Management and Technical Delivery and business stakeholders
- Optimize DevOps workflows not just for speed and stability, but for **cost-efficiency and scalability**


**ROADMAP**
### Month 1-2: Linux, Git & Docker + Cost Fundamentals
- Learn Linux basics: file systems, permissions, user/process/network management
- Bash scripting (cron jobs, AWK, sed) for automation
- Git & GitHub: branching, pull requests, rebasing
- Docker: image creation, container lifecycle, volumes/networks
- **Cost Insight**: Understand local vs. containerized dev environments (resource use, team scalability)
- Project: Dockerize a shell script or micro app and version control it

### Month 2-3: CI/CD & Automation with Cost in Mind
- Build automated pipelines with GitHub Actions and Jenkins
- Integrate testing and deployment stages in workflows
- Understand deployment strategies (blue/green, rolling)
- **Cost Insight**: Analyze build pipeline costs, cloud-hosted runners vs. self-hosted, open-source vs. enterprise tools
- Project: CI/CD pipeline for a demo app

###  Month 3-4: AWS Deep Dive + Infrastructure as Code
- Hands-on with EC2, S3, IAM, Lambda, VPC, and CLI
- Write reusable Terraform code for AWS provisioning
- **Cost Insight**: Understand AWS pricing models, billing dashboard, cost calculators, and budgeting tools (e.g., Budgets, Cost Explorer)
- 🛠️ Project: Deploy a web app to EC2 with infrastructure provisioned by Terraform

###  Month 4-5: Kubernetes & Monitoring
- Kubernetes: pods, deployments, services, scaling basics
- Use Minikube or EKS for deployments
- Add monitoring with Prometheus & Grafana
- **Cost Insight**: Compare self-managed K8s vs. managed services (EKS), cost of observability tooling
- 📊 Project: Deploy a monitored microservice app with resource usage tracked

### Month 5-6: Real-World DevOps System + Strategic Wrap-Up
- Design a full pipeline: Dev → Build → Test → Deploy → Monitor
- Implement versioned IaC, containerized services, and CI/CD
- 🧠 **Cost Insight**: Present total cost of ownership, usage data, and DevOps ROI for your architecture
- 🗂️ Project: Build a capstone DevOps solution and document all decision-making


###  Observability & Monitoring
- Log collection, system monitoring, and alerting
- Prometheus & Grafana dashboards
- Cloud-native monitoring (CloudWatch)

### Soft Skills & Management Layer
- How DevOps connects with Agile and software delivery
- Managing DevOps-focused teams effectively
- Communicating technical progress to stakeholders
